# VM-Quiz Test Vue

## Installation

To install the dependencies for this project, run the following command in the root of the project:

```
npm install
```

## Running the Development Server

To run the development server, use the following command:

```
npm run dev
```

This will start a local development server and open the app in your default browser. The server will also watch for changes in the project files and automatically reload the page as you make updates.
